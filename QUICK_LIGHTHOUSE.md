# 🚀 Быстрый запуск Lighthouse Audit

## Шаг 1: Запустите сервер

В терминале выполните:
```bash
cd /Users/vidmich/Desktop/prac
python3 start_server.py
```

Сервер откроется на http://localhost:8000

## Шаг 2: Откройте в Chrome

1. Откройте Chrome
2. Перейдите на http://localhost:8000
3. Дождитесь полной загрузки страницы

## Шаг 3: Запустите Lighthouse

1. **Откройте DevTools:**
   - Нажмите `F12` или `Cmd+Option+I` (Mac) / `Ctrl+Shift+I` (Windows)
   - Или правый клик → "Inspect"

2. **Перейдите на вкладку Lighthouse:**
   - Если вкладки нет, нажмите `>>` (двойная стрелка) вверху DevTools
   - Выберите "Lighthouse"

3. **Настройте аудит:**
   - ✅ Performance
   - ✅ Accessibility  
   - ✅ Best Practices
   - ✅ SEO
   - Режим: **Navigation** (по умолчанию)

4. **Запустите:**
   - Нажмите кнопку **"Analyze page load"**
   - Дождитесь завершения (30-60 секунд)

## Шаг 4: Просмотрите результаты

Lighthouse покажет:
- 📊 Оценки по каждой категории (0-100)
- 🔴 Критичные проблемы (красные)
- 🟡 Предупреждения (желтые)
- ✅ Что уже хорошо

## Что делать дальше?

1. **Скопируйте результаты** или сделайте скриншот
2. **Исправьте критичные проблемы** (красные)
3. **Улучшите предупреждения** (желтые)
4. **Повторите аудит** после исправлений

---

💡 **Совет:** Lighthouse в Chrome DevTools - самый простой способ, не требует установки дополнительных инструментов!
